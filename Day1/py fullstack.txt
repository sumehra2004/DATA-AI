### 4ï¸âƒ£ Sequence Types

#### List (mutable)

### ðŸ“Œ **List in Python**

---

## 1ï¸âƒ£ Explanation

A **list** in Python is a built-in data type used to **store multiple values in a single variable**.
Lists are:

* **Ordered** (items have a fixed position)
* **Mutable** (you can change items)
* Can store **different data types**
* Allow **duplicate values**

Lists are heavily used in **Data, AI, and real applications** to store and process collections of data.

---

## 2ï¸âƒ£ Syntax

```python

```

---

## 3ï¸âƒ£ Code Example

```python
# Creating a list

# Accessing elements
print(numbers[0])     # 10

# Modifying elements

# Adding elements

# Removing elements

```
